### 概述 ###

对于 Java 锁的分类没有严格意义的规则。我们常说的分类，一般都是依据锁的特性、锁的设计、锁的状态等进行归纳整理出来的。如下使我们常见的所分类：
+ 公平锁/非公平锁
+  可重入锁
+ 独享锁/共享锁
+ 互斥锁/读写锁
+ 分段锁
+ 偏向锁/轻量级锁/重量级锁
+ 自旋锁

### 公平锁/非公平锁 ###
公平锁是指多个线程按照申请锁的顺序获取锁。而非公平锁则是完全没有顺序，随机获取锁，有可能后申请锁的线程会比先申请锁的线程先获取到锁，这就可能会造成饥饿现象或者优先级反转（**饥饿现象和优先级反转在后面详细了解**）。
